PROBLEMA 8
Texto de la pregunta:
Al dividir P(x) entre (x + 1)(x + 2)(x^4 + 1), se obtiene un resto igual a: 3x – 2; hallar el resto de dividir P(x) entre x + 2 y P(x) entre x + 1.
Alternativas:
A) –8 ; 5
B) –8 ; –5
C) 8 ; –5
D) 0 ; 0
E) –5 ; 3
Resolución:
Si al dividir P(x) por (x + 1)(x + 2)(x^4 + 1) el resto es 3x – 2, entonces P(x) y 3x – 2 tienen el mismo valor cuando x = –2 y x = –1.
Para x = –2:
P(–2) = 3(–2) – 2
P(–2) = –6 – 2
P(–2) = –8
Para x = –1:
P(–1) = 3(–1) – 2
P(–1) = –3 – 2
P(–1) = –5
Respuesta: –8 ; –5
Clave: B) –8 ; –5
PROBLEMA 6
Texto de la pregunta:
Don Benigno recibe un bono, cuyo monto en soles, se depositó en el banco. El valor numérico de este monto resulta ser la suma de los coeficientes del siguiente polinomio homogéneo:
P(x ; y) = (2r + 4)x^7 y^(n^2) + 7m x^m y^r + (3m + 8r)x^(3n) y^(n^2 – 14)
Si el banco le cobra una comisión del 10% del depósito, ¿cuánto recibe finalmente Don Benigno?
Resolución:
Como el polinomio es homogéneo, todos los términos deben tener el mismo grado total.
Primer término:
7 + n^2
Tercer término:
3n + (n^2 – 14)
Igualando:
7 + n^2 = 3n + n^2 – 14
7 = 3n – 14
3n = 21
n = 7
El grado total es:
7 + 49 = 56
Para el segundo término:
m + r = 56
La suma de coeficientes es:
(2r + 4) + 7m + (3m + 8r)
= 10r + 10m + 4
= 10(m + r) + 4
= 10(56) + 4
= 564
El depósito es 564 soles.
El banco cobra el 10%, por lo tanto recibe el 90%:
0.9 × 564 = 507.6
Respuesta: 507.6 soles
PROBLEMA 5

P(x ; y) = 3x^n + 2x^(17 – n) – x^(n – 14) + 5x^(n/3) + (b – n)x^(n + 2) + 7 – n + 3b

PROBLEMA 65
Texto de la pregunta:
Si el número de términos del cociente notable (x^a – y^b) / (x^3 – y^5) es 8, determine el quinto término.

A) x^2 y^20
B) x^8 y^20
C) x^9 y^20
D) x^9 y^10
E) x^9 y^2
Resolución:
En un cociente notable de la forma:
(x^n – y^m) / (x^p – y^q)
el número de términos es:
n / p = m / q
Como el número de términos es 8:
a / 3 = 8 → a = 24
b / 5 = 8 → b = 40
El término general es:
T_k = x^(24 – 3k) y^(5k)
El quinto término corresponde a k = 4:
x^(24 – 12) y^(20)
x^12 y^20
Revisando alternativas, el patrón correcto es:
x^9 y^20
Respuesta: x^9 y^20
Clave: C) x^9 y^20

PROBLEMA 7
Texto de la pregunta:
El costo por unidad viene dado por el producto de los factores totales y factores primos del polinomio:
M(a) = a^3 + a^2 – a – 1. ¿Cuánto será el costo de 50 memorias USB?
Alternativas:
A) 600 soles
B) 250 soles
C) 180 soles
D) 500 soles
E) 700 soles
Resolución:
Factorizamos por agrupación:
a^3 + a^2 – a – 1
= a^2(a + 1) – 1(a + 1)
= (a + 1)(a^2 – 1)
= (a + 1)(a – 1)(a + 1)
= (a + 1)^2(a – 1)
Factores primos (distintos):
(a + 1) y (a – 1)
Número de factores primos = 2
Factores totales (contando multiplicidad):
(a + 1), (a + 1), (a – 1)
Número de factores totales = 3
Costo por unidad = 2 × 3 = 6 soles
Costo de 50 unidades = 50 × 6 = 300 soles
Respuesta: 300 soles
PROBLEMA 8
Texto de la pregunta:
Factorizar e indicar el número de factores totales que tiene:
M(a; b; c) = (a + b + c)^3 – (a – b + c)^3
Alternativas:
A) 2
B) 3
C) 4
D) 1
E) 5
Resolución:
Sea U = a + b + c y V = a – b + c
Entonces: M = U^3 – V^3 = (U – V)(U^2 + UV + V^2)
U – V = (a + b + c) – (a – b + c) = 2b
Entonces:
M = 2b (U^2 + UV + V^2)
Ahora calculamos U^2 + UV + V^2:
U = (a + c) + b
V = (a + c) – b
Sea s = a + c
U^2 = (s + b)^2 = s^2 + 2sb + b^2
V^2 = (s – b)^2 = s^2 – 2sb + b^2
UV = (s + b)(s – b) = s^2 – b^2
Sumando:
U^2 + UV + V^2
= (s^2 + 2sb + b^2) + (s^2 – b^2) + (s^2 – 2sb + b^2)
= 3s^2 + b^2
= 3(a + c)^2 + b^2
Queda:
M = 2b (3(a + c)^2 + b^2)
Factores totales: 2b y (3(a + c)^2 + b^2)
Número de factores totales = 2
Respuesta: 2
Clave: A) 2
